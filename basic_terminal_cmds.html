🚀 实战演练工作流程

下面是一个完整的项目下载和运行流程，跟着一步步做：

  1. 看看我现在在哪个文件夹： pwd
  2. 看看当前文件夹里有什么文件： ls
  3. 回到用户主目录（就像回到自己的房间）： cd ~
  4. 创建一个新文件夹，用今天的日期命名： mkdir 20250713
  5. 进入刚才创建的文件夹： cd 20250713
  6. 从GitHub下载我们的Python项目（就像下载一个游戏）：
    git clone https://github.com/aicoder2048/python-basic-keywords
  7. 进入下载好的项目文件夹： cd python-basic-keywords
  8. 安装项目需要的工具和库（就像安装游戏需要的组件）： uv sync
  9. 运行我们的数字猜猜乐游戏！ uv run guess_num_simple.py
  10. 如果以后想更新项目，可以用这个命令获取最新版本： git pull

💡 小贴士

  • 终端命令大小写敏感，要完全按照格式输入
  • 如果命令输错了，按 Ctrl+C 可以取消当前命令
  • 按上箭头键可以重复上一个命令
  • Tab键可以自动补全文件名和文件夹名
  • 遇到问题不要怕，多练习就会熟练啦！
